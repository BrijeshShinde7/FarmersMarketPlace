# 🌾 Farmers Market Place

## 📌 Project Overview
The **Farmers Market Place** is a comprehensive e-commerce platform designed to connect farmers directly with consumers. This project aims to streamline the agricultural supply chain by providing a user-friendly interface for buying and selling farm-fresh produce and other agricultural goods.

---

## 🔗 Repository
The main repository for this project is located at:  
👉 https://github.com/BrijeshShinde7/FarmersMarketPlace.git

---

## 🗂️ Project Structure
The repository contains multiple components of the Farmers Market Place project:

- **Spring Boot Backend** → Java-based backend application  
- **ASP.NET Core Backend** → C#-based backend alternative  
- **React Frontend** → Responsive web interface for users  

All components are organized into separate folders within the main repository.

---

## 🚀 Features

- User Authentication (Admin, Farmer, Customer)  
- Product Management  
- Shopping Cart Functionality  
- Order Processing  
- Payment Integration  
- User Profile Management  
- Admin Dashboard  
- Responsive Design  
- REST API endpoints for:
  - Farmers  
  - Products  
  - Stock details  
  - User interactions  
- CORS support  
- Swagger API documentation

## ⚙️ Getting Started

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/BrijeshShinde7/Online_Vegetable_-_Fruits_Store.git
cd Online_Vegetable_-_Fruits_Store
```

---

## ☕ Spring Boot Backend

- Ensure Java JDK 8+ is installed  
- Navigate to backend folder  

    ```bash
  ./mvnw spring-boot:run
    ```

- Access: http://localhost:9090  

---

## 🔵 ASP.NET Core Backend

- Ensure .NET SDK is installed
  ```bash
    dotnet restore
  ```

- Update connection string in `appsettings.json`  

    ```bash
       dotnet run
    ```

---

## ⚛️ React Frontend

  ```bash  
    npm install
    npm start
```

---

## 📄 API Documentation

Swagger UI available at:

   ```bash 
     /swagger
```

(when backend is running in development mode)
